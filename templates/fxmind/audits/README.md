# Audit reports

Reports from **`/fxmind audit`** go **only** in this folder.

| Save here | Example |
|-----------|---------|
| `.fxmind/audits/<resource>.md` | `.fxmind/audits/garages.md` |

**Do not** create `audit-*.md` in the parent `.fxmind/` directory — that layout is deprecated.

Template (read-only): `.fxmind/audit.template.md`
